# Installation

## Requirements

To install the UITest, [Node.js](https://nodejs.org) environment is required.

## Install from NPM

```bash
$ npm i uitest --save-dev
```

## Sample

```bash
$ git clone https://github.com/macaca-sample/uitest-sample.git --depth=1
$ cd uitest-sample
$ npm i
$ npm run test
```

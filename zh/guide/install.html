# 安装

# 环境需要

要安装 UITest, 你需要首先安装 Node.js (opens new window)，中国用户可以安装 cnpm (opens new window) 加快 NPM 模块安装速度。

# 命令行安装

$ npm i uitest --save-dev

# 运行示例

$ git clone https://github.com/macaca-sample/uitest-sample.git --depth=1
$ cd uitest-sample
$ npm i
$ npm run test
